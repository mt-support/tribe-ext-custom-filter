Filter displayed events by time of day.

== Description ==

Create a custom filter for Filter Bar that narrows down the displayed events by time of day. This is a working example created from https://theeventscalendar.com/knowledgebase/k/creating-custom-filters-for-filter-bar/ and can be altered to suit your needs.
